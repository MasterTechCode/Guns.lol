# Assets Directory

Please add your assets to this directory:

## Required Files

1. **Background Video** (`videos/bg.mp4`)
   - Place your background video here
   - Recommended: 1920x1080 or higher resolution
   - Format: MP4 (H.264 codec recommended)
   - Keep file size reasonable for web (<10MB if possible)

2. **Avatar Image** (`images/avatar.png`)
   - Your profile picture
   - Recommended: Square image (500x500px or larger)
   - Format: PNG or JPG
   - Will be displayed as circular avatar

3. **Music File** (`music/song.mp3`)
   - Your background music track
   - Format: MP3
   - Recommended bitrate: 128-192 kbps for web

## Optional

You can also add:
- Alternative background images
- Additional music tracks
- Custom icons or graphics

## Note

The application will work without these files, but the background video, avatar, and music player features will not function properly until you add your assets.
